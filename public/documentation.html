⚓️ Documentation API – Port de Plaisance Russell

Vue d’ensemble

  • Base URL : https://port-plaisance-api-6egt.onrender.com
  • Technologies : Node.js, Express, MongoDB Atlas, JWT
  • Authentification : via token JWT

Tutoriel d’utilisation

1. Enregistrement

POST /api/auth/register
{
  "email": "jeanluc@russell.fr",
  "password": "russell2025",
  "name": "Jean-Luc"
}

2. Connexion

POST /api/auth/login
{
  "email": "jeanluc@russell.fr",
  "password": "russell2025"
}

3. Utilisation du token

Authorization: Bearer <JWT_TOKEN>

Exemples de requêtes

Catways

  • GET /api/catways – Lister tous les catways
  • GET /api/catways/:id – Détails d’un catway
  • POST /api/catways – Créer un catway
  • PUT /api/catways/:id – Modifier un catway
  • PATCH /api/catways/:id – Modifier partiellement
  • DELETE /api/catways/:id – Supprimer un catway

Réservations

  • GET /api/catways/:id/reservations – Lister les réservations
  • GET /api/catways/:id/reservations/:idReservation – Détails d’une réservation
  • POST /api/catways/:id/reservations – Ajouter une réservation
  • DELETE /api/catways/:id/reservations/:idReservation – Supprimer une réservation
